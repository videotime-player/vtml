Especificación del formato VideoTime Media List (VML1) y VideoTime List Group (VLG1)

Versión 1.0.0joegardner

¿Qué son las listas en formato VideoTime Media List (VML1)?

Las listas en formato VideoTime Media List (VML1) es un formato de lista multimedia en línea basado en XML que permite almacenar contenido multimedia alojado en la red y distribuirlo a dispositivos Android a través del cliente VideoTime

¿Qué son los grupos de listas en formato VideoTime List Group (VML1)?

Los grupos de listas en formato VideoTime List Group (VLG1) son similares a las listas en formato VideoTime Media List (VML1), pero a diferencia de las listas en formato VideoTime Media List (VML1), los grupos de listas en formato VideoTime List Group (VLG1) permiten almacenar varias listas en formato VideoTime Media List (VML1) para acceder fácilmente a ellas, evitando tener que agregar cada una de las listas almacenadas al gestor de listas del cliente VideoTime

Funcionamiento de las listas (VML1) y grupos de listas (VLG1)

El cliente VideoTime al cargar la lista o grupo de listas desde Internet, se conecta al servidor y descarga la lista o grupo de listas al cliente, una vez descargada, se procede a procesar los datos de la lista o grupo de listas y finalmente el cliente VideoTime comenzará a mostrar los elementos de la lista o grupo de listas

Ventajas de las listas en formato VideoTime Media List (VML1) frente a las listas en formato M3U

·         Actualización dinámica, al cargarse desde Internet

·         Capacidad de utilizar portadas en tamaño de caja de DVD

·         Carga más rápida, al estar basado en XML

·         Capacidad de incluir una descripción para cada elemento multimedia

Estructura de los archivos de lista

Extensión de archivo: .vml1

Etiquetas:

<vtml-list> Definición de la lista

<vtml-mediaitem> Definición del elemento multimedia, se coloca dentro de <vtml-list>

<vtml-itemname> Definición del nombre del elemento, se coloca dentro de <vtml-mediaitem>

<vtml-itemcover> Definición del caratula del elemento, se coloca dentro de <vtml-mediaitem>

<vtml-itemurl> Definición de la dirección URL multimedia del elemento, se coloca dentro de <vtml-mediaitem>

<vtml-itemdescription> Definición de la descripción del elemento, se coloca dentro de <vtml-mediaitem>

<vtml-itemcategory> Definición de la categoría del elemento, se coloca dentro de <vtml-mediaitem>

Estructura de código:

<?xml version="1.0" encoding="UTF-8"?>

<vtml-list>

       <vtml-mediaitem>

             <vtml-itemname>[NAME]</vtml-itemname>   

             <vtml-itemcover>[COVER]</vtml-itemcover>

             <vtml-itemurl>[URL]</vtml-itemurl>

             <vtml-itemdescription>[DESCRIPTION]</vtml-itemdescription>

             <vtml-itemcategory>[CATEGORY]</vtml-itemcategory>

       </vtml-mediaitem>

</vtml-list>   

Estructura de los archivos de grupo de listas

Extensión de archivo: .vlg1

Etiquetas:

<vtml-group> Definición del grupo de listas

<vtml-listitem> Definición del elemento de lista, se coloca dentro de <vtml-group>

<vtml-itemname> Definición del nombre del elemento, se coloca dentro de <vtml-listitem>

<vtml-itemcover> Definición del caratula del elemento, se coloca dentro de <vtml-listitem>

<vtml-itemurl> Definición de la dirección URL de lista del elemento, se coloca dentro de <vtml-listitem>

<vtml-itemdescription> Definición de la descripción del elemento, se coloca dentro de <vtml-listitem>

<vtml-itemcategory> Definición de la categoría del elemento, se coloca dentro de <vtml-listitem>

Estructura de codigo:

<?xml version="1.0" encoding="UTF-8"?>

<vtml-group>

       <vtml-listitem>

             <vtml-itemname>[NAME]</vtml-itemname>   

             <vtml-itemcover>[COVER]</vtml-itemcover>

             <vtml-itemurl>[URL]</vtml-itemurl>

             <vtml-itemdescription>[DESCRIPTION]</vtml-itemdescription>

             <vtml-itemcategory>[CATEGORY]</vtml-itemcategory>

       </vtml-listitem>

</vtml-group>
